Munro

A GitHub Pages site to host data from Hill Lists. The Hill Lists app uses data from the Database of British and Irish Hills

The maps are generated via pandas and Folium locally from manually updated data files.

Individual Maps

These show data for individuals - green symbols indicate climbed Munros, blue are still to be climbed.

Overlap Maps

Manually generated maps for 2 and 3 person groups showing hills all members in each group still have to do.
